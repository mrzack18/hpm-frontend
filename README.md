# hpm-frontend
hpm-frontend adalah frontend untuk Himatif Project Management (HPM), aplikasi manajemen proyek mirip Trello. Dibangun dengan Svelte + Vite, frontend ini menyediakan antarmuka interaktif untuk mengelola proyek, board, task, user, serta sistem invite, sehingga kolaborasi tim menjadi lebih mudah dan terstruktur.
